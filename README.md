# Weather App 🌤️

A responsive **React + TypeScript** weather application that fetches real-time weather data using the **OpenWeatherMap API**.  

Features include:
- Current location detection
- Search for any city
- Dynamic weather icons based on conditions (Sunny, Cloudy, Rain, Mist, etc.)
- Temperature display in Celsius
- Humidity & wind speed info
- Styled with **styled-components** for a modern and clean interface
- Fully responsive design
